# Support

## Getting Help

- **Bug or feature request?** Open an [issue on GitHub](https://github.com/token-hoarder/docdrop/issues)
- **Question about usage?** Check the [README](README.md) first — the Quick Start and Troubleshooting sections cover the most common problems
- **Something else?** Open an issue and describe what you need

## Before Opening an Issue

1. Search [existing issues](https://github.com/token-hoarder/docdrop/issues) to avoid duplicates
2. Include your OS, Python version, and the exact error message or behavior you're seeing
3. If it's an OCR issue, also include which engine you're using (Tesseract or Surya)
