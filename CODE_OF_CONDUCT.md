# Code of Conduct

## Our Pledge

We are committed to making participation in this project a welcoming experience for everyone, regardless of background or experience level.

## Our Standards

**Be kind.** Constructive criticism is welcome; personal attacks are not.

**Be helpful.** If someone is confused, help them understand instead of dismissing the question.

**Be respectful.** Disagreement is fine — disrespect is not.

## Reporting Issues

If you experience or witness unacceptable behavior, open an issue or contact the maintainer directly via GitHub.

## Enforcement

Project maintainers may remove, edit, or reject any contributions that violate this Code of Conduct.
